# Miras Oyunu

[![CI](https://github.com/rudukan/miras/actions/workflows/ci.yml/badge.svg)](https://github.com/rudukan/miras/actions/workflows/ci.yml)

Türkiye'nin makro koşullarında 1M USD mirası işleten, gerçek zamanlı piyasa verili tarayıcı tabanlı finansal simülasyon oyunu. **[miras-one.vercel.app](https://miras-one.vercel.app)**

## Geliştirme

```bash
npm install
npm run dev     # http://localhost:5173
npm run test    # Vitest (domain + API unit testleri)
npm run check   # svelte-check (TypeScript strict)
```

SvelteKit 2 · Svelte 5 runes · TypeScript strict · Tailwind CSS · Vitest

> Mimari ve tasarım kararları için `memory.md`, sprint planları için `docs/superpowers/` dizinine bakın.
